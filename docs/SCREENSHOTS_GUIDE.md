## Screenshot Guidelines

### Technical Requirements

**Resolution**:
- Desktop: 1920x1080 or higher
- Mobile: 375x667 (iPhone) or 768x1024 (iPad)

**Format**:
- PNG format for clarity
- Compress for web (use tools like TinyPNG)
- Target file size: < 500KB per image

**Browser**:
- Use Chrome or Firefox
- Ensure consistent UI rendering
- Disable browser extensions that modify UI

### Content Guidelines

**Sample Content**:
- Use realistic, professional content
- Avoid lorem ipsum
- Use actual SEO-related topics
- Keep content appropriate and professional

**Scores**:
- Show realistic scores (not all 100s)
- Demonstrate various states (good, ok, problem)
- Use actual analyzer messages

**Annotations**:
- Use clear, readable fonts
- Use arrows and labels sparingly
- Maintain consistent annotation style
- Use colors that contrast with UI

### Capture Process

1. **Prepare Environment**:
   - Fresh WordPress installation
   - MeowSEO plugin installed and activated
   - Sample content prepared
   - Browser window sized appropriately

2. **Capture Screenshots**:
   - Use browser's built-in screenshot tool or Snagit/Greenshot
   - Capture full interface or specific sections as needed
   - Ensure no personal information visible
   - Check for UI consistency

3. **Edit and Annotate**:
   - Crop to relevant area
   - Add annotations as specified
   - Ensure text is readable
   - Maintain consistent style

4. **Optimize**:
   - Compress images
   - Verify quality
   - Test in documentation
   - Ensure fast loading

## Using Screenshots in Documentation

### Markdown Syntax

```markdown
![Alt text](./images/meowseo-sidebar-overview.png)
*Caption: MeowSEO sidebar showing real-time analysis scores*
```

### Best Practices

1. **Always include alt text** for accessibility
2. **Add captions** to explain what's shown
3. **Reference in text** before showing image
4. **Keep images relevant** to surrounding content
5. **Update regularly** to match current UI

### Example Usage

```markdown
## Getting Started

After installing MeowSEO, you'll see the analysis panel in the Gutenberg sidebar:

![MeowSEO Sidebar Overview](./images/meowseo-sidebar-overview.png)
*The MeowSEO panel displays real-time SEO and Readability scores*

The panel shows two main scores:
- **SEO Score**: Measures search engine optimization
- **Readability Score**: Measures content clarity
```

## Accessibility Considerations

### Alt Text Guidelines

**Good alt text**:
- Describes what's shown
- Mentions key UI elements
- Includes relevant scores/states
- Keeps it concise (< 125 characters)

**Examples**:

✅ Good:
```markdown
![MeowSEO sidebar showing SEO score of 85 (green) and Readability score of 72 (green)](./images/meowseo-sidebar-overview.png)
```

❌ Bad:
```markdown
![Screenshot](./images/meowseo-sidebar-overview.png)
```

### Screen Reader Considerations

- Ensure alt text provides context
- Don't rely solely on images to convey information
- Provide text descriptions alongside images
- Use captions to supplement visual information
